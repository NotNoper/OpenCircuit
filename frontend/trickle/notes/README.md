# Open Circuit Project Notes

This document tracks the development and updates of the Open Circuit website.

## Project Structure
- `index.html` / `app.js`: Landing page.
- `playground.html` / `playground-app.js`: Component simulator.
- `assembly.html` / `assembly-app.js`: Camera-based assembly guide.
- `datasheets.html` / `datasheets-app.js`: Component specification lookup.
- `components/`: Shared UI components.

## Maintenance Rules
- Check this file for updates whenever significant architectural changes are made.
- Update `trickle/assets` if new external images are added.
- Update `trickle/rules` if new development constraints are established.
